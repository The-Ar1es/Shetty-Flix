# 🎬 Shetty-Flix

A front-end clone of Netflix's landing page, built with HTML and CSS.
Submitted as my entry for the **SnW Hackathon**.

---

## 📌 About

Shetty-Flix recreates the look and feel of Netflix India's landing page —
including the hero section, feature highlights, and overall layout —
using only HTML and CSS.

---

## 🗂️ Project Structure
```
Shetty-Flix/
├── index.html    # Main page structure
├── style.css     # All styling
└── images/       # Assets sourced from netflix.com/in
```
## 🛠️ Built With

- HTML5
- CSS3
